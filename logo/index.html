🐢 NeoLogo Instructions

🎯 Quick Start

1. Write code in the editor on the left.
2. Click "Run" or press Ctrl+Enter.
3. Watch the turtle draw on the canvas!

🐢 Turtle Commands

  • HIDETURTLE or HT - Hides the turtle.
  • SHOWTURTLE or ST - Shows the turtle.

📐 Basic Movement

  • FD 100 - Moves forward 100 pixels.
  • BK 50 - Moves backward 50 pixels.
  • RT 90 - Turns right 90 degrees.
  • LT 45 - Turns left 45 degrees.
  • HOME - Returns to the center, facing up.

🖊️ Pen Control

  • PU - Lifts the pen (moves without drawing).
  • PD - Lowers the pen (draws while moving).
  • SETCOLOR red - Changes the pen color.
  • SETWIDTH 5 - Changes the pen thickness.

🎨 Available Colors

red, blue, green, yellow, orange, purple, pink, brown, black, white, gray, cyan, magenta

🔄 Loops & Procedures

REPEAT 6 [ FD 100 RT 60 ]  # Draws a hexagon

TO star
  REPEAT 5 [ FD 100 RT 144 ]
END

star  # Calls the procedure

🔢 Variables

MAKE "size 50
FD :size  # Use the variable with a colon

⌨️ Keyboard Shortcuts

  • Ctrl+Enter - Runs the program.
  • Escape - Closes this help menu.
